Accidental on tied-to note

When a note with an accidental is tied across a bar line, the standard convention is to not print the accidental on the tied-to note, and Mup uses that convention. If you really want to have the accidental printed, you can specify a slur rather than a tie. If you want the MIDI output to be correct, you can use an ifdef to use slur for printing and tie for MIDI. 

1: 2c;e& ifdef MIDI ~ else <> endif ;
bar

1: 1e&;
bar
